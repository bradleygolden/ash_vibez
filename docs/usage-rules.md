# Usage Rules

Package authors provide `usage-rules.md` files with LLM guidance.

## Direct Access

From GitHub:
```
https://raw.githubusercontent.com/<org>/<package>/main/usage-rules.md
https://raw.githubusercontent.com/<org>/<package>/main/usage-rules/<topic>.md
```

From installed deps:
```
deps/<package>/usage-rules.md
deps/<package>/usage-rules/<topic>.md
```

## Using the Library

For automatic consolidation and hexdocs search, install `usage_rules`:

https://hexdocs.pm/usage_rules/readme.html
