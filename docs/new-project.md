# New Ash Project

```bash
# One-time setup
mix archive.install hex igniter_new
mix archive.install hex phx_new  # if using Phoenix

# Create project
mix igniter.new <app_name> --install ash,<extensions> --with phx.new
```

See [Extensions](extensions.md) for available packages.

After creating your project, fetch [usage rules](usage-rules.md) for your dependencies.
