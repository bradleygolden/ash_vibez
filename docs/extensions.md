# Ash Extensions

## Available Packages

| Package | Purpose |
|---------|---------|
| `ash_postgres` | PostgreSQL database |
| `ash_sqlite` | SQLite database |
| `ash_phoenix` | Phoenix integration, forms, LiveView |
| `ash_json_api` | REST API (JSON:API spec) |
| `ash_graphql` | GraphQL API |
| `ash_authentication` | User auth (passwords, OAuth, magic links) |
| `ash_authentication_phoenix` | Phoenix UI for auth |
| `ash_oban` | Background jobs |
| `ash_admin` | Auto-generated admin UI |
| `ash_archival` | Soft deletion |
| `ash_paper_trail` | Audit logs |
| `ash_state_machine` | State machines |
| `ash_cloak` | Attribute encryption |
| `ash_money` | Money/currency types |
| `ash_double_entry` | Double-entry accounting |

## Discover More

Search hex.pm for `ash_*` packages:
```
https://hex.pm/packages?search=ash&sort=recent_downloads
```

## Adding to Existing Project

```bash
mix igniter.install <package>
```
